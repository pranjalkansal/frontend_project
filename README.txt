A web project containing both frontend as well as backend.

** Front-end **

Frontend contains the HyperText Markup Language (HTML), Cascading Style Sheets (CSS) and JavaScript (JS) code to make a website good looking
and responsive.

The JavaScript framework used is Angular Js which is currently one of the most popular javascript framework.

** Back-end **

Backend is based upon Node js that is currently the most popular server side javascript language.

Node js is further used with Hapi js which is one of the popular framework to write Node js code.

DataBase: A NoSQL database is used in this project i.e. MongoDB.
